# Task 5 - Exploratory Data Analysis (EDA)

## 📊 Dataset
Titanic Dataset (Kaggle)

## 🎯 Objective
To perform Exploratory Data Analysis (EDA) on the Titanic dataset and extract meaningful insights using Python.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Steps Performed
1. Loaded dataset using Pandas  
2. Checked dataset structure using `.info()`  
3. Generated statistical summary using `.describe()`  
4. Handled missing values (Age column)  
5. Created visualizations:
   - Survival count plot  
   - Survival by gender  
   - Age distribution  
   - Age vs survival (boxplot)  
   - Fare distribution  
   - Survival by passenger class  
   - Correlation heatmap  
   - Pairplot  

## 📈 Key Insights
- Majority of passengers did not survive  
- Females had a higher survival rate than males  
- Younger passengers had slightly better survival chances  
- First-class passengers had higher survival rates  
- Most passengers paid lower fares  
- Fare shows slight positive correlation with survival  

## 📌 Conclusion
EDA helps in understanding data patterns, relationships, and trends, which is essential before applying machine learning models.
